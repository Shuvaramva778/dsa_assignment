# Reverse Traversal of Singly Linked List

## (a) Data Structure Definition
A singly linked list is implemented using a structure containing `data` and a pointer `next`.

## (b) Functions Implemented
- `push()` – Inserts a new node at the beginning.
- `reversePrint()` – Prints the linked list in reverse order using recursion.

## (c) main() Overview
The `main()` function creates a linked list and calls the reverse traversal function.

## (d) Sample Output
Reverse Traversal: 10 20 30
