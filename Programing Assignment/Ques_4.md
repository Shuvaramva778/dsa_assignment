# Doubly Linked List Implementation

## (a) Data Structure Definition
Each node of the doubly linked list contains `data`, a pointer to the previous node (`prev`), and a pointer to the next node (`next`).

## (b) Functions Implemented
- `insertAfter()` – Inserts a node after a given node.
- `deleteNode()` – Deletes a specified node.
- `display()` – Displays the list elements.

## (c) main() Overview
The `main()` function demonstrates insertion and deletion in a doubly linked list.

## (d) Sample Output
List: 10 20 30  
After Deletion: 10 30
